MKPC CT Project Track Guide

    Track Type:

    Map color coding:

    Every map image of a track has lines of some color. Here is what the colors mean:

    Blue: A normal shortcut
    Red: A no item shortcut (nisc)
    Purple: A slower route that has more items (for bagging)

    Bill spots:

    A bill spot can be an extension or a shortcut you can take with the bullet. All bullet extensions need to be used in last place because the time you spend in a bullet depends on the placement you used it in.
